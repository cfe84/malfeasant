Welcome to My Blog

Posted on July 12, 2025

This is the actual blog content, served from a separate static directory. This content is protected by the honeypot system but won't count towards rate limiting for legitimate users.

The honeypot system differentiates between:

  • Blog content (this folder) - Real content for visitors
  • Honeypot content (public folder) - Demo/trap content
  • Dashboard - Administrative interface
  • API endpoints - System management

How the Rate Limiting Works

Posted on July 12, 2025

The system applies intelligent rate limiting:

  • API and dashboard requests are excluded from rate limits
  • Only HTML content requests count towards limits
  • Legitimate visitors can browse freely
  • Bots and scrapers get blocked automatically

You can test the blocking by adding ?scramble to any URL.
